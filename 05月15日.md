## 第一题：寻找旋转排序数组中的最小值 https://leetcode.cn/problems/find-minimum-in-rotated-sorted-array/?envType=daily-question&envId=2026-05-15

**思路：二分查找。首先数组具有单调性，就得想到二分；然后这个单调数组进行了单调旋转，那么它依旧具有单调性，只需要每次mid和最后一个元素比较大小，就可以确定最小值在哪一边，然后不断二分得到答案**

**代码：**
```python
class Solution:
    def findMin(self, nums: List[int]) -> int:
        n = len(nums)
        l, r = 0, n - 1
        while l <= r:
            mid = (l + r) >> 1
            if nums[mid] > nums[-1]:
                l = mid + 1
            else:
                r = mid - 1
        return nums[r + 1]
```
