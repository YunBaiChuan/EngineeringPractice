**思路：这题我们不难发现，要使得最终的 mex 尽可能大，需要我们使用一个贪心策略。**

**若希望最终的 mex 为 $x+1$，那么需要构造一个式子  
$$\sum_{i=1}^{x} \alpha_i \cdot i$$  
要使得求和尽量大，只需令  
$$\alpha_i = x - i + 1$$  
那么最终的求和就是  
$$\sum_{i=1}^{x} (x - i + 1) \cdot i$$  
化简式子得到
$$\sum_{i=1}^{x} (x + 1) - i^2$$
继续化简式子得到
$$\sum_{i=1}^{x} (x + 1) - \sum_{i=1}^{x} i^2$$
接续化简式子得到
$$\frac{x(x+1)^2}{2} - \frac{x(x+1)(2x+1)}{6}$$
化简后得到  
$$\sum_{i=1}^{x} (x - i + 1) \cdot i = \frac{x(x+1)(x+2)}{6}$$  
然后接下来就是二分答案的事情。**

**代码：**
```python
import sys
input = lambda: sys.stdin.readline().strip()

import math
inf = 10 ** 18

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def GMI():
    return map(lambda x: int(x) - 1, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''
数学推导+二分答案，注意最终ans需要+1，因为mex是没有出现的最小非负整数
我们二分答案的是计算的x，最终mex应该是x + 1
'''

def solve():
    m = II()

    def check(mid):
        return (mid + 1) * mid * (mid + 2) // 6 <= m

    l, r = 1, 10 ** 18
    while l <= r:
        mid = (l + r) >> 1
        if check(mid):
            l = mid + 1
        else:
            r = mid - 1
    print(r + 1)


t = II()
for _ in range(t):
    solve()
```
