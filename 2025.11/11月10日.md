## 第一题：3542.将所有元素变为0的最少操作次数 https://leetcode.cn/problems/minimum-operations-to-convert-all-elements-to-zero/description/?envType=daily-question&envId=2025-11-10

**思路：他其实是一个单调栈，对于"小大小"这种情况来说，当遇到第二个小的情况时，要进行单调栈的弹出，即"大"要进行一次操作，最终还要把单调栈中所有非0的元素进行一次操作，就是最终的答案。**

**代码：**
```python
class Solution:
    def minOperations(self, nums: List[int]) -> int:
        ans = 0
        st = []
        for x in nums:
            while st and x < st[-1]:
                st.pop()
                ans += 1
            # 如果 x 与栈顶相同，那么 x 与栈顶可以在同一次操作中都变成 0，x 无需入栈
            if not st or x != st[-1]:
                st.append(x)
        return ans + len(st) - (st[0] == 0)  # 0 不需要操作
```

## 第二题：小红的好矩阵 https://ac.nowcoder.com/acm/contest/121300/C

**思路：题目说的矩形面积为2的情况其实就只有两种，一个是1 * 2还有一个是2 * 1，这两个情况相互独立，并且长和宽也是相互独立的，那么特判一下不满足的情况之后，就可以直接用数学公式解决了。**

**代码：**
```python
import sys
input = lambda: sys.stdin.readline().strip()

import math
inf = math.inf

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def GMI():
    return map(lambda x: int(x) - 1, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''
1 * 2 == 2; 2 * 1 == 2
两种情况
'''

def solve():
    n, m = MII()
    if n * m < 2:
        print(0)
        return
    print((m - 1) * n + m * (n - 1))

t = 1
for _ in range(t):
    solve()
```

## 第三题：小红嫁接 https://ac.nowcoder.com/acm/contest/121300/D

**思路：思维题，我们仔细分析他的操作过程，最终达到链其实就是每个节点的度为1或者2，因此对于原图来说，我们计算出每个节点的度数，如果其度数>2那么就把多余的部分给拆下来，即deg - 2，那其实最终拆下来的这些节点，每个度数都是1或者2，最终把他们连接起来就可以得到一条长链了。那其实我们的结果就是$ans = sum(deg - 2) \ if \ deg > 2 \ else \ 0$。主要是从度这方面入手思考。**
