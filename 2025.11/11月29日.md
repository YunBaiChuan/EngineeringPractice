**思路：其实就是概率论推式子。首先s表示从16个粒子中选出4个为铜/银/金的组合数ans就分为两个部分，一个是整体16个粒子都为铜，为p ** 16，直接用pow即可还有一个部分是12个铜粒子 + 4个银粒子/金粒子，直接对应的概率和组合数相乘即可**

**代码：**
```python
import sys
input = lambda: sys.stdin.readline().strip()

import math
inf = 10 ** 18

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def GMI():
    return map(lambda x: int(x) - 1, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''
需要12个铜粒做稿子，然后就是工作台，都是需要四个粒子材料
从贪心的角度出发，如果铜粒有16个那就用，否则银粒、金粒有4个那就用

哦哦，概率论推式子。首先s表示从16个粒子中选出4个为铜/银/金的组合数
ans就分为两个部分，一个是整体16个粒子都为铜，为p ** 16，直接用pow即可
还有一个部分是12个铜粒子 + 4个银粒子/金粒子，直接对应的概率和组合数相乘即可
'''

def solve():
    a, b, c = LII()
    s = (16 * 15 * 14 * 13) / (4 * 3 * 2 * 1)
    p = a / 16
    q = b / 16
    r = c / 16
    ans = pow(p, 16) + pow(p, 12) * (pow(q, 4) + pow(r, 4)) * s
    print(f"{ans:.10f}")


t = II()
for _ in range(t):
    solve()
```
