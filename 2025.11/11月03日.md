## 第一题：C-小红的区间查询 https://ac.nowcoder.com/acm/contest/120553/C 

**思路：这是一个比较经典的trick，相离区间查找具体点的区间标号。其实就是用排序(左端点)+二分，我们查找刚好$>X_i$的区间，然后再-1就可以得到$<= X_i$的区间，接着我们判断当前点是否在这个区间内即可。**

**需要额外注意的是，二分查找多维区间$[a, b, c]$时，要注意每个位置的效果，比如说查找$[l, inf, \_]$就表示以l为标准进行二分，如果l相同的话，就看第二个位置。此处为什么是$inf$？我们可以取l相同的时候，如果此时第二位置是-inf的话，此处区间的第二位置必然更大，那么最终就会停留在当前位置，最终-1的话导致区间位置错乱。反之，如果是inf，他就会继续往后，达成正确结果。**

**优化策略，对于二分查找多维区间来说，非常慢。一般的，我们单独提取出l，做一个单维度的二分，速度会更快。**


**代码：**
```python
import bisect
import sys

import math

def I():
    return input()

    return int(input())

def MII():
    return map(int, input().split())

def GMI():
    return map(lambda x: int(x) - 1, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''

'''

def solve():
    n, q = MII()
    cs = []
    for i in range(1, n + 1):
        l, r = MII()
        cs.append([l, r, i])
    cs.sort()
    for _ in range(q):
        x = II()
        p = bisect.bisect_right(cs, [x, inf, _])
        if p == 0:
            print(-1)
        else:
            p -= 1
            print(cs[p][2] if cs[p][0] <= x <= cs[p][1] else -1)

t = 1
for _ in range(t):
    solve()
```

## 第二题：D-小红的区间相交 https://ac.nowcoder.com/acm/contest/120553/D

**思路：初看不容易有思路，建议从必要条件分析入手，也就是在满足条件下，它具有什么性质。先分析包含关系，如果按照左端点排序，与右端点排序得到的结果不同，那说明有包含关系；反之则没有。因此，我们可以先判断有无包含关系。接着分析两两区间相交关系，其实只需要判断排序后最左边和最右边区间是否相交即可，如果相交的话，那么剩下的区间就都会被压缩到这两个区间之间，实现两两相交。**

**代码：**
```python
import sys
input = lambda: sys.stdin.readline().strip()

import math
inf = math.inf

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def GMI():
    return map(lambda x: int(x) - 1, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''

'''

def solve():
    n = II()
    a = [LII() for _ in range(n)]
    b = a.copy()
    a.sort()
    b.sort(key=lambda x:(x[1], x[0]))
    if a != b:
        print("No")
        return

    def check(p, q):
        l1, r1 = p[0], p[1]
        l2, r2 = q[0], q[1]
        if r1 < l2 or l1 > r2:
            return False
        return True

    if check(a[0], a[-1]):
        print("Yes")
    else:
        print("No")


t = II()
for _ in range(t):
    solve()
```

## 第三题：C. Earning on Bets https://codeforces.com/problemset/problem/1979/C

**思路：首先注意到当$min(a) > n$时，我们只需要全部取$1$即可；否则，就要进行以下的公式推理：**

### 数学推导

1. 原条件（题意）
   令总的硬币数量为$S$，那么对于每个赌注$K_i$需要满足以下条件：

   $$
   K_i * (分配数) > S
   $$

   接着移项一下得到：

   $$
   分配数 > \frac{S}{K_i}
   $$

   然后左右两边取$sum$
   
   $$
   \sum_{i=1}^{n} \frac{S}{k_i} < S
   $$

   然后左右两边同时去掉$S$，得到

   $$
   \sum_{i=1}^{n} \frac{1}{k_i} < 1
   $$

2. 通分：取公共分母 $D = \mathrm{lcm}(k_1,k_2,\dots,k_n)$（关键）
   $$
   \sum_{i=1}^{n} \frac{1}{k_i}
   = \frac{1}{D}\sum_{i=1}^{n} \frac{D}{k_i}
   $$

3. 不等式两边同乘 $D$（$D>0$）
   $$
   \frac{1}{D}\sum_{i=1}^{n} \frac{D}{k_i} < 1
   \quad\Longleftrightarrow\quad
   \sum_{i=1}^{n} \frac{D}{k_i} < D
   $$

4. 令 $x_i = \dfrac{D}{k_i}$（必为正整数），则
   $$
   \sum_{i=1}^{n} x_i < D
   $$

只要上式成立，$\{x_i\}$ 即为一组可行解；否则无解，输出 $-1$。

**代码：**
```python
import sys
input = lambda: sys.stdin.readline().strip()

import math
inf = math.inf

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def GMI():
    return map(lambda x: int(x) - 1, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''
纯数学推式子，然后加点观察
'''

def solve():
    n = II()
    a = LII()
    if min(a) > n:
        ans = [1] * n
        print(*ans)
    else:
        z = math.lcm(*a)
        s = 0
        for x in a:
            s += z // x
        if s < z:
            ans = [z // x for x in a]
            print(*ans)
        else:
            print(-1)

t = II()
for _ in range(t):
    solve()
```
