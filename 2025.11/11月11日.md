## 第一题：474.一和零 https://leetcode.cn/problems/ones-and-zeroes/description/?envType=daily-question&envId=2025-11-11

**思路：比较有技巧性，如果说我们数组中只有0，然后让我们选恰好m个0，那就很容易可以看出来是恰好背包问题。这一题是最多m个0，以及n个0，那他其实是一个三维背包问题，我们要多一个维度的参数如下：$dfs(i, j, k)$表示从$0$到$i - 1$选择一些字符串，使得$0$数量<j并且$1$的数量<k，然后就可以在我们二维背包上面进行扩展，写记忆化搜索会更简单一些。**

**代码：**
```python
class Solution:
    def findMaxForm(self, strs: List[str], m: int, n: int) -> int:
        cnt0 = [s.count('0') for s in strs]
        @cache  
        def dfs(i: int, j: int, k: int) -> int:
            if i < 0:
                return 0
            res = dfs(i - 1, j, k)  # 不选 strs[i]
            cnt1 = len(strs[i]) - cnt0[i]
            if j >= cnt0[i] and k >= cnt1:
                res = max(res, dfs(i - 1, j - cnt0[i], k - cnt1) + 1)  # 选 strs[i]
            return res

        return dfs(len(strs) - 1, m, n)
```
