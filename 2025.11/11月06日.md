## 第一题：3607.电网维护 https://leetcode.cn/problems/power-grid-maintenance/?envType=daily-question&envId=2025-11-06

**思路：电网相互连接，每次查询要找电网中在线的最小id电站，那可以想到维护一个电网的并查集，并且每次可以查找到里面的最小在线id电站，有两种方法：1.最小懒删除堆。2.SortedList有序列表。用2的话好写一点，具体来说就是先创一个并查集，之后通过连边进行并查集的合并，接着初始化SortedList表，把每个电网中的最小在线id电站放进去，接着就可以回答询问了。**

**代码：**
```python
'''
并查集应该是，维护每个电网中的最小编号在线电站
'''
class UnionFind:
    def __init__(self, n: int):
        # 一开始有 n 个集合 {0}, {1}, ..., {n-1}
        # 集合 i 的代表元是自己，大小为 1
        self._fa = list(range(n))  # 代表元
        self._size = [1] * n  # 集合大小
        self.cc = n  # 连通块个数

    # 返回 x 所在集合的代表元
    # 同时做路径压缩，也就是把 x 所在集合中的所有元素的 fa 都改成代表元
    def find(self, x: int) -> int:
        fa = self._fa
        # 如果 fa[x] == x，则表示 x 是代表元
        if fa[x] != x:
            fa[x] = self.find(fa[x])  # fa 改成代表元
        return fa[x]

    # 判断 x 和 y 是否在同一个集合
    def is_same(self, x: int, y: int) -> bool:
        # 如果 x 的代表元和 y 的代表元相同，那么 x 和 y 就在同一个集合
        # 这就是代表元的作用：用来快速判断两个元素是否在同一个集合
        return self.find(x) == self.find(y)

    # 把 from 所在集合合并到 to 所在集合中
    # 返回是否合并成功
    def merge(self, from_: int, to: int) -> bool:
        x, y = self.find(from_), self.find(to)
        if x == y:  # from 和 to 在同一个集合，不做合并
            return False
        self._fa[x] = y  # 合并集合。修改后就可以认为 from 和 to 在同一个集合了
        self._size[y] += self._size[x]  # 更新集合大小（注意集合大小保存在代表元上）
        # 无需更新 _size[x]，因为我们不用 _size[x] 而是用 _size[find(x)] 获取集合大小，但 find(x) == y，我们不会再访问 _size[x]
        self.cc -= 1  # 成功合并，连通块个数减一
        return True

    # 返回 x 所在集合的大小
    def get_size(self, x: int) -> int:
        return self._size[self.find(x)]  # 集合大小保存在代表元上

class Solution:
    def processQueries(self, c: int, connections: List[List[int]], queries: List[List[int]]) -> List[int]:
        uf = UnionFind(c + 1)
        for u, v in connections: # 合并u -> v
            uf.merge(u, v)
            
        st = [SortedList() for _ in range(c + 1)] # 初始化有序集合，便于后面找最小id
        for i in range(1, c + 1):
            st[uf.find(i)].add(i)
        ans = []
        
        for a, x in queries:
            fa = uf.find(x)
            if a == 1:
                if x in st[fa]:
                    ans.append(x)
                elif len(st[fa]):
                    ans.append(st[fa][0])
                else:
                    ans.append(-1)
            else:
                st[fa].discard(x) # 这里的删除用discard，如果没有目标删除元素，不会报错。而用remove的话就会报错
        return ans 
```

## 第二题：E - Shift String https://atcoder.jp/contests/abc430/tasks/abc430_e

**思路：题目大意是说有两个字符串A、B，我们可以对A进行类似于队列的操作，即先进先出，问能否通过最少次数得到B。有两种邪修做法：1.hash值，可以用单hash，如果无法通过，就用多hash降低hash冲突概率。2.观察大法，我们其实可以直接拼两个a，然后find查找b即可，不过提交要用cpython，底层是c，处理字符串更快，用pypy会超时。不过pypy对于大数据，以及多重循环的处理会比cpython更好。**

**代码：**
```python
import sys
from collections import Counter

input = lambda: sys.stdin.readline().strip()

import math
inf = math.inf

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def GMI():
    return map(lambda x: int(x) - 1, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''
邪修做法，因为只能从前面移除，然后添加到后面，因此可以直接在原有a
的基础上再拼接一个a，再用find查找b，刚好返回的索引就是操作次数。

不过要注意使用cpython来提交，因为cpython底层是c优化的，处理起
字符串就非常快，不会超时。而pypy主要是在多重循环，数值计算方面比较快
'''

def solve():
    a = I()
    b = I()
    print((a * 2).find(b))


t = II()
for _ in range(t):
    solve()
```

## 第三题：F - Back and Forth Filling https://atcoder.jp/contests/abc430/tasks/abc430_f

**思路：题目大意是说，给你一个$n - 1$长度的只包含"L"和"R"的字符串$S$，表示第$i + 1$数字在第$i$数字的$s[i]$边，问最终每个$C_i$位置有多少种填法？这题需要找必要条件，可以发现我们需要四个数组来记录每个位置i前后连续的"L"和"R"个数，最终区间$[rp[i] + ls[i + 1], n - 1 - lp[i] - rs[i + 1]]$内都应该$C_i+1$，对于多个区间的$+1$操作来说，需要用到差分，最终求个前缀和即可。**

**代码：**
```python
import sys
from itertools import accumulate

input = lambda: sys.stdin.readline().strip()

import math
inf = math.inf

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def GMI():
    return map(lambda x: int(x) - 1, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''
必要条件分析，差分
'''

def solve():
    n = II()
    s = '.' + I() + '.'
    lp, rp = [0] * (n + 1), [0] * (n + 1) # 求连续的正向l, r
    for i in range(1, n):
        if s[i] == 'L': lp[i] = lp[i - 1] + 1
        if s[i] == 'R': rp[i] = rp[i - 1] + 1
    ls, rs = [0] * (n + 1), [0] * (n + 1) # 求连续的逆向l, r
    for i in range(n - 1, 0, -1):
        if s[i] == 'L': ls[i] = ls[i + 1] + 1
        if s[i] == 'R': rs[i] = rs[i + 1] + 1
    bk = [0] * (n + 1)
    for i in range(n):
        l, r = 0, n - 1
        l += (rp[i] + ls[i + 1])
        r -= (lp[i] + rs[i + 1])
        bk[l] += 1
        bk[r + 1] -= 1
    acc = list(accumulate(bk))
    print(*acc[:-1])

t = II()
for _ in range(t):
    solve()
```
