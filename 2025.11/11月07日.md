## 第一题：2528.最大化城市的最小电量 https://leetcode.cn/problems/maximize-the-minimum-powered-city/?envType=daily-question&envId=2025-11-07

**思路：大致来说由两部分拼合而成：1.最大化最小值的二分。2.差分数组，涉及到区间+1问题。因此，我们首先差分预处理出每个城市的电力，接着就可以用二分来最大化最小电力，也就是写check函数。对于电力不够的城市来说，我们就要用批准我们建造的$K$个电站进行补全，如果不够直接返回$False$；否则我们就贪心的进行补全，什么意思？也就是把电站建造到$i + r$位置，这样电站就能覆盖到$[i, i + 2r]$范围，能覆盖最多城市，这样把多建造的城市也是放在差分数组中，然后累加求和，就可以实现check函数，最终二分求出ans**

**代码：**
```python
class Solution:
    def maxPower(self, stations: List[int], R: int, K: int) -> int:
        n = len(stations)
        """将每个城市的电量处理出来，差分数组实现"""
        flash = [0] * n
        diff = [0] * (n + 1)
        for i, x in enumerate(stations):
            diff[max(0,i - R)] += x
            diff[min(n,i + R + 1)] -= x 
        s = 0
        for i in range(n):
            s += diff[i]
            flash[i] = s
        """检查是否每个城市在K个额外电站的条件下电量大于等于lim"""
        def check(lim: int) -> bool:
            cnt = 0
            diff = [0] * (n + 1)
            # 建一个电站，可以让[l,l + 2 * r]被覆盖到，贪心的讲就是放在l + r位置，那么左右两边就是[l, l + 2 * r]都可以被覆盖到
            s = 0
            for i, x in enumerate(flash):
                s += diff[i]
                if s + x < lim:
                    num = lim - (s + x)
                    cnt += num 
                    if cnt > K:
                        return False
                    diff[i] += num 
                    diff[min(n, i + 2 * R + 1)] -= num 
                    s += num
            return True 
        """二分模板求右边界"""
        l = 0
        r = sum(stations) + K
        while l <= r:
            mid = (l + r) >> 1
            if check(mid):
                l = mid + 1
            else:
                r = mid - 1
        return r
```
