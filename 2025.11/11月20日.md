## 第一题：来点gcd https://www.nowcoder.com/practice/16d153d47b334b0f8cc507b70a2e2473?channelPut=tracker2

**思路：我们要找到是否存在一个集合满足$gcd = x$，那就可以直接预处理来枚举$gcd$，把$1 <= x <= n$的$gcd$都给枚举一遍，然后找a数组中有的元素进行$gcd$计算，最终回答询问即可**

**代码：**
```python
import sys
input = lambda: sys.stdin.readline().strip()

import math
inf = 10 ** 18

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def GMI():
    return map(lambda x: int(x) - 1, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''
gcd枚举预处理，但是python会狠狠超时
'''

def solve():
    n, m = MII()
    a = LII()
    st = set(a)
    ans = [False] * (n + 1)

    for x in range(1, n + 1):
        g = 0
        for d in range(x, n + 1, x):
            if d in st:
                g = math.gcd(g, d)
        if x == g:
            ans[x] = True

    for _ in range(m):
        x = II()
        print("YES" if ans[x] else "NO")

t = II()
for _ in range(t):
    solve()
```

## 第二题：757.设置交集大小至少为2 https://leetcode.cn/problems/set-intersection-size-at-least-two/description/?envType=daily-question&envId=2025-11-20

**思路：用的是栈+二分，并且这里的d可以从2扩展到k，比较困难，先放着**

**代码：**
```python
class Solution:
    def intersectionSizeTwo(self, intervals: List[List[int]]) -> int:
        intervals.sort(key=lambda interval: interval[1])
        # 栈中保存闭区间左右端点，栈底到栈顶的区间长度的和
        st = [(-2, -2, 0)]  # 哨兵，保证不和任何区间相交
        for start, end in intervals:
            _, r, s = st[bisect_left(st, (start,)) - 1]
            d = 2 - (st[-1][2] - s)  # 去掉运行中的时间点
            if start <= r:  # start 在区间 st[i] 内
                d -= r - start + 1  # 去掉运行中的时间点
            if d <= 0:
                continue
            while end - st[-1][1] <= d:  # 剩余的 d 填充区间后缀
                l, r, _ = st.pop()
                d += r - l + 1  # 合并区间
            st.append((end - d + 1, end, st[-1][2] + d))
        return st[-1][2]
```
