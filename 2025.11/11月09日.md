## 第一题：2169.得到0的操作数 https://leetcode.cn/problems/count-operations-to-obtain-zero/description/?envType=daily-question&envId=2025-11-09

**思路：模拟是很简单的，但当两个数差值非常大的时候，模拟就可能超时了，此时就需要用到辗转相除。注意到我们每次进行操作的时候，操作次数是$x // y$，完成之后$x$就变成了$x \% y$，此时的$x$是要比$y$更小的，因此还需要进行两者的交换，这里有两种写法：1.递归。2.循环。**

**代码：**
```python
class Solution:
    def countOperations(self, num1: int, num2: int) -> int:
        # 模拟写法
        ans = 0 
        while num1 and num2:
            if num1 > num2:
                num1 -= num2
            else:
                num2 -= num1 
            ans += 1
        return ans 

class Solution:
    def countOperations(self, x: int, y: int) -> int:
        # 辗转相除法
        ans = 0
        while y > 0:
            ans += x // y 
            x, y = y, x % y
        return ans
```
