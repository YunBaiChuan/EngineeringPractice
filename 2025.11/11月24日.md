## 第一题：小苯的左右移动 https://ac.nowcoder.com/acm/contest/122724/D

**思路：这题出得挺好的，比较经典的优化，再加点思维。大概就是整体处理左右移动，并且记录其中的最大右移数，因为我们左移是只需要添加0就可以的，而右移可能会吞掉我们原先位置的1。最终完成之后，就可以用位运算来得到原始k值，然后就可以讨论s值的正负，以此来得到最终的k。最终这里的位运算也是要优化一下的，不能直接用位运算进行移动，会得到超大整数，处理会超时。因此我们直接用$a * b\ \%\  mod = (a\ \% \mod * b\ \%\ mod)\ \%\ mod$这个公式，但是对于有除法的情况，我们需要用小费马定理得到它对应的逆元，比如说$a = 1 / 2 ^ t$，那么它的逆元就是$a^{-1} = pow(2^t, mod - 2,\ mod)$，注意这里的mod必须是质数，不过一般来说都是质数，最终就可以通过了**

**代码：**
```python

    n, k = MII()
    a = LII()

    mn = inf
    s = 0
    for i, x in enumerate(a, 1):
        if i & 1:
            s += x
        else:
            s -= x
        if s < mn:
            mn = s

    if mn < 0: # 最关键优化，不需要构造出字符串，一个来回位移动就行，而且时间和空间都更优
        k = (k >> abs(mn)) << abs(mn)

    if k == 0:
        print(0)
        return

    p = k % mod # p * q % mod的优化，这里算是比较标准的位运算优化，可以提取成板子了
    if s > 0:
        xs = pow(2, s, mod)
        print(p * xs % mod)
    elif s < 0:
        xs = pow(2, abs(s), mod)
        inx_xs = pow(xs, mod - 2, mod)
        print(p * inx_xs % mod)
    else:
        print(p)

t = II()
for _ in range(t):
    solve()
```

## 第二题：能量辐射 https://www.nowcoder.com/practice/04c83aedd3e043879ed4464baae0b564?channelPut=tracker2

**思路：也是一道比较裸的单调栈+贡献法板子题，就是板子有点忘了。简单说来，对于此题，如果遇到当前元素$v >= a[st[-1]]$，就进行弹出，其实就是一个从下往上的递减栈，就根据题意来就行。具体代码就是一个left和一个right数组，其中left初始化为-1，right初始化为n，left正向移动，right逆向移动，也就是分别左寻找和右寻找。找的过程中就是先弹出，再判断栈是否非空来填left、right，最终用贡献法算一个ans数组，最终取max即可**

**代码：**
```python
import sys
input = lambda: sys.stdin.readline().strip()

import math
inf = 10 ** 18

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def GMI():
    return map(lambda x: int(x) - 1, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''
单调栈
'''

def solve():
    n = II()
    hv = [LII() for _ in range(n)]
    a = [h for h, v in hv]
    b = [v for h, v in hv]

    # 单调栈板子
    left = [-1] * n
    st = []
    for i, v in enumerate(a):
        while st and v >= a[st[-1]]:
            st.pop()
        if st:
            left[i] = st[-1]
        st.append(i)

    st.clear()
    right = [n] * n
    for i in range(n - 1, -1, -1):
        x = a[i]
        while st and x > a[st[-1]]:
            st.pop()
        if st:
            right[i] = st[-1]
        st.append(i)

    ans = [0] * n
    for i, (l, r) in enumerate(zip(left, right)):
        if 0 <= l <= n - 1:
            ans[l] += b[i]
        if 0 <= r <= n - 1:
            ans[r] += b[i]

    print(max(ans))

t = 1
for _ in range(t):
    solve()
```
