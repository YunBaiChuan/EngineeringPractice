## 第一题：2257.统计网格图中没有被保卫的格子数 https://leetcode.cn/problems/count-unguarded-cells-in-the-grid/description/?envType=daily-question&envId=2025-11-02

**思路：正向思考比较困难，我们可以用逆向思维：求每个能被守卫的格子，最终$O(m * n)$进行一次遍历找没有被守卫的格子即可。但是代码要写得好看点，就需要用到两个技巧：1.提前标记墙和守卫为True，这样后面就不会再去判断墙或者守卫，而造成多余的时间复杂度。2.使用DIRE方向数组，能够大大简化代码长度。**

**代码：**
```python
class Solution:
    def countUnguarded(self, m: int, n: int, guards: List[List[int]], walls: List[List[int]]) -> int:
        guarded = [[False] * n for _ in range(m)]
        wall_set = set(map(tuple, walls))
        guard_set = set(map(tuple, guards))

        # 标记守卫和墙本身为“已处理”
        for x, y in guard_set:
            guarded[x][y] = True
        for x, y in wall_set:
            guarded[x][y] = True

        directions = [(-1, 0), (1, 0), (0, -1), (0, 1)]

        for x, y in guard_set:
            for dx, dy in directions:
                nx, ny = x + dx, y + dy
                while 0 <= nx < m and 0 <= ny < n:
                    if (nx, ny) in wall_set or (nx, ny) in guard_set:
                        break
                    guarded[nx][ny] = True
                    nx += dx
                    ny += dy

        # 统计未被守护的格子
        unguarded = 0
        for i in range(m):
            for j in range(n):
                if not guarded[i][j]:
                    unguarded += 1

        return unguarded
```

## 第二题：C - Odd One Subsequence https://atcoder.jp/contests/abc429/tasks/abc429_c

**思路：题目大意是说选出两个相同元素，以及一个不同元素，问有多少种方案数。这个其实可以用组合数学的知识来解决，首先我们要用$cnt_v$统计每个元素的数量，接着我们对每个$cnt_v$选两个元素，那么还有一个不同的元素就是$n - cnt_v$，因此每个$cnt_v$的贡献就是$C(cnt_v, 2) * (n - cnt_v)$，整理一下就是$cnt_v * (cnt_v - 1) \ //\  2 * (n - cnt_v)$，其实就是乘法原理，最终把每个$cnt_v$的贡献相加，得到的就是最终结果**

**代码：**
```python
import sys
input = lambda: sys.stdin.readline().strip()

import math
inf = math.inf

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def GMI():
    return map(lambda x: int(x) - 1, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''
组合数学，乘法原理 + 加法原理。先选出两个相同的数，即C(cntv, 2)
然后再选出一个不是v的数，即n - cntv。最后再用前面说的乘法原理+加法原理即可
'''

def solve():
    n = II()
    nums = LII()
    cnt = [0] * (n + 1)

    for x in nums:
        cnt[x] += 1

    ans = 0
    for x in cnt:
        ans += x * (x - 1) // 2 * (n - x)

    print(ans)

t = 1
for _ in range(t):
    solve()
```

## 第三题：D - On AtCoder Conference https://atcoder.jp/contests/abc429/tasks/abc429_d

**思路：题目大意是说，我们有一个周长为$m$的圆周，然后$A_i$表示每个人从$0$位置开始，顺时针的距离。我们目标是求解从$0$到$m - 1$索引出发，遇到刚好$>=C$的人数$X_i$之和。这里是通过展平之后，并且把每个位置看作为一个人，此时c作为人数，其实就可以看成距离了，之后再通过二分、贡献法就可以得到最终结果**

**代码：**
```python
import bisect
import sys
input = lambda: sys.stdin.readline().strip()

import math
inf = math.inf

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def GMI():
    return map(lambda x: int(x) - 1, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''
感觉比较复杂，先放着，晚上写总结的时候模拟一下
'''

def solve():
    n, m, c = LII()
    nums = LII()

    nums.sort() # 首先排序，因为移动是有序的
    for i in range(n * 2): # 接着模拟两轮移动，将循环展平，其实就是每轮结束后其距离+m
        nums.append(nums[i] + m)

    ans = 0 # 相当于每个位置就是一个人，那么c就变成了距离
    for i in range(1, n + 1):
        v = nums[i + c - 1]
        p = bisect.bisect_right(nums, v)
        ans += (p - i) * (nums[i] - nums[i - 1]) # 这里(p - i)为人数，然后(nums[i] - nums[i - 1])表示能够作为起点的数量，整体就是一个区间的贡献

    print(ans)


t = 1
for _ in range(t):
    solve()
```
