## 第一题：模板 https://ac.nowcoder.com/acm/contest/123263/A

**思路：贪心，诈骗。由于只能在末尾进行字符的删除和添加，所以说它就不是子序列dp。这样想，我们进行操作的时候，只能是头对齐，用修改操作，最后再把末尾给删了，就是最优解了。**

**代码：**
```python
import sys
input = lambda: sys.stdin.readline().strip()

import math
inf = 10 ** 18

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def GMI():
    return map(lambda x: int(x) - 1, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''
诈骗题，最小是假的，操作是固定的
'''

def solve():
    m, n = MII()
    s = I()
    t = I()
    ans = 0
    for x, y in zip(s, t):
        if x != y:
            ans += 1
    print(ans + abs(m - n))

t = 1
for _ in range(t):
    solve()
```

## 第二题：I题是个签到题 https://ac.nowcoder.com/acm/contest/123263/B

**思路：模拟即可，可以熟练下range负数写法，以及math.ceil的*向上取整，注意题意理解**

理解出问题了

## 第三题：牛牛战队的秀场 https://ac.nowcoder.com/acm/contest/123263/C

**思路：主要就是知道圆的内接多边形的边长计算公式，然后就是左右找最小值即可。其公式为：$d = 2 * r * sin(\pi/n)$，他是用正弦公式推的。**

**代码：**
```python
import sys
input = lambda: sys.stdin.readline().strip()

import math
inf = 10 ** 18

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def GMI():
    return map(lambda x: int(x) - 1, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''
多边形数学公式
'''

def solve():
    n, r = MII()
    i, j = MII()
    if i > j:
        i, j = j, i
    a = 2 * r * math.sin(math.pi / n)
    p = j - i; q = n - p
    print(fmin(a * p, a * q))

t = 1
for _ in range(t):
    solve()
```

## 第四题：Enjoy the game https://ac.nowcoder.com/acm/contest/123263/D

**思路：手玩样例，打表观察，注意到其实可以用dp解决，但是n非常大，那就说明必然存在某种数学结论，能够解决本题，其实就是当$n = 2^x$时，后手$Alice$必胜，其他情况下都是先手$Bob$必胜。**

**代码：**
```python
import sys
input = lambda: sys.stdin.readline().strip()

import math
inf = 10 ** 18

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def GMI():
    return map(lambda x: int(x) - 1, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''
打表猜结论，太奇妙了。我只猜测了当n为2，4时，后手alice会赢，但是
n为8的情况我没模拟对，他其实也是alice赢，那我们就可以大胆猜测2 ** i
时，后手alice会赢，就是正确的。这里只给出打表代码，注意pypy3里面的缓存器
该用lru_cache()
'''

from functools import lru_cache

def solve():
    @lru_cache()
    def dfs(remaining, last):
        if remaining == 0:
            return False
        max_take = min(last, remaining)
        if last == 0:  # 第一步，last=0 表示“虚拟”状态
            max_take = remaining - 1
        for take in range(1, max_take + 1):
            if not dfs(remaining - take, take):
                return True
        return False

    for n in range(2, 101):
        winner = "Bob" if dfs(n, 0) else "Alice"
        print(f"{n:3d} -> {winner}")

t = 1
for _ in range(t):
    solve()
```
