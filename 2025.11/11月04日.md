## 第一题：3318.计算子数组的x-sum I https://leetcode.cn/problems/find-x-sum-of-all-k-long-subarrays-i/description/?envType=daily-question&envId=2025-11-04

**思路：数据范围比较小，考虑比较暴力的方法。直接写定长滑动窗口的hash表模拟即可。这里想到一个比较好的结论，那就是以当前位置$i$为基础，左边取长度$k$的位置就是$i - k + 1$，反之右边取长度$k$的位置就是$i + k - 1$，两个部分是对称的，方便记忆。**

**代码：这里写定长滑动窗口的板子**
```python
cnt = Counter()
for r, v in enumerate(nums):
    cnt[v] += 1 # 入窗口
    if r < k - 1:
        continue 

    # 各种窗口内元素处理，按照题意来写

    cnt[nums[l - k + 1]] -= 1 # 出窗口
    if cnt[nums[l - k + 1]] == 0:
        del cnt[nums[l - k + 1]]
```
