## 第一题：1611.使整数变为0的最少操作次数 https://leetcode.cn/problems/minimum-one-bit-operations-to-make-integers-zero/description/?envType=daily-question&envId=2025-11-08

**思路：数学找规律，推式子。首先逆向思维，把$n$到$0$逆向思考成$0$到$n$，方便找规律。接着思考特殊情况，也就是从$0$变到$2^k$的情况，并且定义$f(n)$表示从$0$到$n$需要的最少操作次数，可以推理出$f(2^k) = 2*f(2^{k - 1}) + 1$，转化一下式子可以得到$f(2^k) = 2^{k + 1} - 1$。再来考虑一般情况，其实就是$f(2^{k - 1}) - f(n - 2^{k - 1})$，带入前面的特殊化式子就有$ans = 2^k - 1 - f(n - 2^{k - 1})，对于这种有重复性的式子来说，我们采用递归来写，当然也是可以翻译成递推的。$**

**代码：**
```python
class Solution:
    def minimumOneBitOperations(self, n: int) -> int:
        if n == 0:
            return 0
        k = n.bit_length()
        return (1 << k) - 1 - self.minimumOneBitOperations(n - (1 << (k - 1)))
```

## 第二题：字段乘积 https://ac.nowcoder.com/acm/contest/120453/C

**思路：很简单，就是定长滑动窗口，但是要注意0的情况，需要进行特殊处理。我们额外用一个zero变量来统计0的个数，然后把0情况在定长滑动窗口中进行额外判断，包括入窗口，处理窗口，出窗口。**

**代码：**
```python
import sys
input = lambda: sys.stdin.readline().strip()

import math
inf = math.inf

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def GMI():
    return map(lambda x: int(x) - 1, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''
特殊处理0情况的定长滑动窗口，注意出窗口的方法，这里采用了费马小定理求逆元然后进行乘法
'''

MOD = 998244353

def solve():
    n, k = MII()
    a = [x % MOD for x in LII()]

    ans = 0
    s = 1
    zero = 0  # 当前窗口 0 的个数，特殊处理

    for r, v in enumerate(a):
        if v == 0:
            zero += 1
        else:
            s = s * v % MOD

        if r < k - 1:
            continue

        if zero == 0:  # 只有无 0 才更新
            ans = fmax(ans, s)
        else:
            ans = fmax(ans, 0)  # 有 0 乘积就是 0

        # 移出左端
        if a[r - k + 1] == 0:
            zero -= 1
        else: # 小费马定理乘逆元出窗口，相当于除法
            s = s * pow(a[r - k + 1], MOD - 2, MOD) % MOD


    print(ans)

t = 1
for _ in range(t):
    solve()
```

## 第三题：字段异或 https://ac.nowcoder.com/acm/contest/120453/D

**思路：非常经典的前缀异或和、找两两相同数**

**代码：**
```python
import sys
from collections import defaultdict
from email.policy import default

input = lambda: sys.stdin.readline().strip()

import math
inf = math.inf

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def GMI():
    return map(lambda x: int(x) - 1, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''
经典前缀异或和 + 找两两相同数
'''

def solve():
    n = II()
    a = LII()
    pre = [0] * (n + 1)
    for i in range(n):
        pre[i + 1] = pre[i] ^ a[i]

    d = defaultdict(int)
    ans = 0
    for x in pre:
        ans += d[x]
        d[x] += 1
    print(ans)

t = 1
for _ in range(t):
    solve()
```
