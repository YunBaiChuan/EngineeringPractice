## 第一题：2872.可以被K整除连通块的最大数目 https://leetcode.cn/problems/maximum-number-of-k-divisible-components/description/?envType=daily-question&envId=2025-11-28

**思路：最大化连通块的数目，等价于最大化删除的边数加一。直接进行dfs，如果遇到能够删除的边，即权重s的为k的倍数，否则就dfs下去找子树权重s为k的倍数，然后进行边的删除。**

**代码：**
```python
class Solution:
    def maxKDivisibleComponents(self, n: int, edges: List[List[int]], values: List[int], k: int) -> int:
        g = [[] for _ in range(n)]
        for x, y in edges:
            g[x].append(y)
            g[y].append(x)

        # 返回子树 x 的点权和
        def dfs(x: int, fa: int) -> int:
            s = values[x]
            for y in g[x]:
                if y != fa:  # 避免访问父节点
                    # 加上子树 y 的点权和，得到子树 x 的点权和
                    s += dfs(y, x)  
            nonlocal ans
            ans += s % k == 0
            return s

        ans = 0
        dfs(0, -1)
        return ans
```

## 第二题：迷途之家的大贤者 https://www.nowcoder.com/practice/019336efe38b4ba982e6eea8b42d2d42?channelPut=tracker2

**思路：其实就是小红第一次删除时，只保留首位、末尾元素，其中的最大字典序就是最终答案。这个性质主要是通过题目说的最终字符串非空，观察得来的。**

**代码：**
