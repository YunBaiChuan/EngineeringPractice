## 第一题：牛牛的DRB迷宫 https://ac.nowcoder.com/acm/contest/120452/A

**思路：网格图dp的变式，比较经典的网格图dp其实就是所有位置都为'B'，也就是每个位置都可以往下或者往右走。而对于本题来说，有的位置只能够单向走，因此递推式子也会发生相应的改变：**

**对于$s[i][j] == D$时，有$dp[i + 1][j] = (dp[i + 1][j] + dp[i][j])\ \%\ mod$
对于$s[i][j] == R$时，有$dp[i][j + 1] = (dp[i][j + 1] + dp[i][j])\ \%\ mod$
当$s[i][j] == B$时，这两个都要进行更新**

**代码：**
```python
import sys
input = lambda: sys.stdin.readline().strip()

import math
inf = math.inf

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def GMI():
    return map(lambda x: int(x) - 1, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''
网格图dp变式，状态及边界发生改变
'''

mod = 10 ** 9 + 7

def solve():
    n, m = MII()
    s = [list(I()) for _ in range(n)]
    dp = [[0] * (m + 2) for _ in range(n + 2)]
    dp[1][1] = 1
    for i in range(1, n + 1):
        for j in range(1, m + 1):
            if s[i - 1][j - 1] == 'R':
                dp[i][j + 1] = (dp[i][j + 1] + dp[i][j]) % mod
            elif s[i - 1][j - 1] == 'D':
                dp[i + 1][j] = (dp[i + 1][j] + dp[i][j]) % mod
            else:
                dp[i][j + 1] = (dp[i][j + 1] + dp[i][j]) % mod
                dp[i + 1][j] = (dp[i + 1][j] + dp[i][j]) % mod
    print(dp[n][m])

t = 1
for _ in range(t):
    solve()
```

## 第二题：牛牛的K合因子数 https://ac.nowcoder.com/acm/contest/120452/B

**思路：首先可以用质数筛把$1$到$n$的合数给标出来，然后就可以对每一个合数算其k因子个数，并且存到ans数组中，便于后续的$O(1)$回答**

**代码：**
```python
import sys
input = lambda: sys.stdin.readline().strip()

import math
inf = math.inf

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def GMI():
    return map(lambda x: int(x) - 1, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''
质数筛，预处理
'''

def solve():
    n, m = MII()
    queries = [II() for _ in range(m)]

    is_prime = [True] * (n + 1)
    is_prime[0] = is_prime[1] = False
    for i in range(2, isqrt(n) + 1):
        if is_prime[i]:
            for j in range(i*i, n+1, i):
                is_prime[j] = False

    composite_factor_count = [0] * (n + 1)
    for i in range(1, n + 1):
        cnt = 0
        for d in range(1, isqrt(i) + 1):
            if i % d == 0:
                if d > 1 and not is_prime[d]: # i因子
                    cnt += 1
                other = i // d # 另一个因子
                if other != d and other > 1 and not is_prime[other]:
                    cnt += 1
        composite_factor_count[i] = cnt

    ans = [0] * (n + 1)
    for i in range(1, n + 1):
        k = composite_factor_count[i]
        if k <= n:
            ans[k] += 1

    for k in queries:
        print(ans[k] if 0 <= k <= n else 0)

t = 1
for _ in range(t):
    solve()
```

## 第三题：牛牛的Link Power https://ac.nowcoder.com/acm/contest/120452/C

**思路：题目描述其实很简单，就是求解所有'1'位置的距离之和，但直接暴力的话，其时间复杂度为$O(n^2)$，因此我们要进行公式的推到，最重要的是固定j位置，然后算j的贡献。**

### 推导：所有下标对距离之和

设有序列 $\{p_0,p_1,\dots ,p_{k-1}\}$，需求
\[
\sum_{0\le i<j<k}(p_j-p_i).
\]

固定 $j$，把内层对 $i$ 的求和展开：
\[
\sum_{i=0}^{j-1}(p_j-p_i)
=\sum_{i=0}^{j-1}p_j-\sum_{i=0}^{j-1}p_i
=j\cdot p_j-\mathrm{prefix}[j],
\]
其中 $\mathrm{prefix}[j]=\sum_{i=0}^{j-1}p_i$。

于是总和为
\[
\boxed{\sum_{j=1}^{k-1}\bigl(j\cdot p_j-\mathrm{prefix}[j]\bigr).}
\]

**代码：**
```python
import sys
input = lambda: sys.stdin.readline().strip()

import math
inf = math.inf

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def GMI():
    return map(lambda x: int(x) - 1, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''
式子转化，前缀和、贡献法优化（固定j位置，算i < j时的贡献）
'''

mod = 10 ** 9 + 7

def solve():
    n = II()
    s = I()
    pos = [i for i, ch in enumerate(s) if ch == '1']

    k = len(pos)
    if k < 2:
        print(0)
        return

    prefix = [0] * (k + 1)
    for i in range(k):
        prefix[i + 1] = (prefix[i] + pos[i]) % mod

    ans = 0
    for j in range(1, k):
        cnt = j
        val = pos[j]
        sum_prev = prefix[j]
        ans = (ans + cnt * val - sum_prev) % mod

    print(ans % mod)

t = 1
for _ in range(t):
    solve()
```

## 第四题：516.最长回文子序列 https://leetcode.cn/problems/longest-palindromic-subsequence/description/?envType=problem-list-v2&envId=dlexqabk

**思路：经典区间$dp$题目，对于此类题目来说，我们通常定义$dp[i][j]$表示区间$i$到$j$之间，满足条件。那对于此题来说，就是最长的回文子序列，有两种状态如下：**

**当$s[i] == s[i]$时，$dp[i][j] = dp[i + 1][j - 1] + 2$，此处$+ \ 2$表示左右两边相同，那么回文序列长度就是$+ \ 2$；
当$s[i] \ != s[j]$时，$dp[i][j] = max(dp[i + 1][j], dp[i][j + 1])$**

**至于翻译成$dp$时候的枚举，我们就看'负顺正逆'即可，对于$i$来说为$+$，那就外层逆向枚举，而对于$j$来说为$-$，那就内层顺序枚举。**

**代码：**
```python
class Solution:
    def longestPalindromeSubseq(self, s: str) -> int:
        # @cache 
        # def dfs(i, j):
        #     if i > j:
        #         return 0 
        #     if i == j:
        #         return 1 
        #     if s[i] == s[j]:
        #         return dfs(i + 1, j - 1) + 2 
        #     return max(dfs(i + 1, j), dfs(i, j - 1))
        # return dfs(0, len(s) - 1)

        # n = len(s)
        # dp = [[0] * n for _ in range(n)]
        # for i in range(n - 1, -1, -1):
        #     dp[i][i] = 1 
        #     for j in range(i + 1, n):
        #         if s[i] == s[j]:
        #             dp[i][j] = dp[i + 1][j - 1] + 2 
        #         else:
        #             dp[i][j] = max(dp[i + 1][j], dp[i][j - 1])
        # return dp[0][-1]

        n = len(s)
        dp = [0] * n 
        for i in range(n - 1, -1, -1):
            dp[i] = 1 
            pre = 0
            for j in range(i + 1, n):
                tmp = dp[j]
                dp[j] = pre + 2 if s[i] == s[j] else max(dp[j], dp[j - 1])
                pre = tmp 
        return dp[-1]
```

## 第五题：让字符串成为回文串的最少插入次数 https://leetcode.cn/problems/minimum-insertion-steps-to-make-a-string-palindrome/description/?envType=problem-list-v2&envId=dlexqabk

**思路：有两种方法解决。方法一：直接板子上面改一下状态转移方程如下：**

**当$s[i] == s[i]$时，$dp[i][j] = dp[i + 1][j - 1]$，表示不需要添加；
当$s[i] \ != s[j]$时，$dp[i][j] = min(dp[i + 1][j], dp[i][j + 1]) + 1$ 表示需要添加一个字符了，接着具体是添加在左边还是右边，我们不太清楚，因此就取两个中的$min$即可**

**方法二：本质就是$n - mx$，我们通过举例可以发现，本题的本质其实就是自身的长度$-$最长回文子序列的长度，他其实是通过枚举回文中心，然后推出要最长回文子序列，接着进行对齐，非回文地方就补字符。**

**本题是可以在任意位置进行字符的插入，如果改成只能在末尾插入呢？那此题就变成一个贪心了，我们要枚举左端点，找到左端点与最右边所能形成的最长回文串，这样左端点左边没有匹配的部分，就可以右边末尾插入来形成回文串，这也是曾经做过的题目。**

**代码：**
```python
class Solution:
    def minInsertions(self, s: str) -> int:
        # n = len(s)
        # @cache 
        # def dfs(i, j):
        #     if i >= j:
        #         return 0 
        #     if s[i] == s[j]:
        #         return dfs(i + 1, j - 1)
        #     return min(dfs(i + 1, j), dfs(i, j - 1)) + 1
        # return dfs(0, len(s) - 1)

        n = len(s)
        @cache 
        def dfs(i, j):
            if i > j:
                return 0 
            if i == j:
                return 1 
            if s[i] == s[j]:
                return dfs(i + 1, j - 1) + 2 
            return max(dfs(i + 1, j), dfs(i, j - 1))
        return n - dfs(0, len(s) - 1)
```
