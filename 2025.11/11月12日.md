## 第一题：2654.使数组所有元素变为1的最少操作次数 https://leetcode.cn/problems/minimum-number-of-operations-to-make-all-array-elements-equal-to-1/description/?envType=daily-question&envId=2025-11-12

**思路：比较容易想到的就是观察性质+暴力，首先如果整体的$gcd$不等于1的话，那说明数组不管怎么操作，每个位置必然>1，因此返回-1。否则，我们可以统计其中相邻$gcd$为1的个数，如果存在的话，我们再判断一下原数组中1的数量，就可以得到$ans = n - cnt1$。如果不存在相邻gcd为1的情况，那说明我们需要找到最少的额外操作次数，使得相邻$gcd$为1。那就可以预处理出一个$tmp$数组，用于存放相邻$gcd$的值，之后就用两个$for$来找$gcd$为1的最近距离(本题数据量很小)，其实就是我们的最小操作次数，此时$ans = n + md$。**

**代码：**
```python
class Solution:
    def minOperations(self, nums: List[int]) -> int:
        n = len(nums)
        if gcd(*nums) != 1:
            return -1

        tmp = []
        for i in range(n - 1):
            tmp.append(gcd(nums[i], nums[i + 1]))
            
        cnt = tmp.count(1)
        if not cnt:
            md = inf # 加上可以产生gcd(nums[i], nums[i + 1])为1的最少操作次数
            for i in range(len(tmp)):
                for j in range(i):
                    if gcd(tmp[i], tmp[j]) == 1:
                        md = min(md, i - j)
            return n + md 

        cnt1 = nums.count(1)
        if cnt1 > 0:
            return n - cnt1 
        else:
            return n

```

## 第二题：C - Robot Factory https://atcoder.jp/contests/abc431/tasks/abc431_c

**思路：非常经典的排序双指针，有点像归并分治思想中，merge部分的有序双指针处理，有序的目的是为了更好的处理数据。**

**代码：**
```python
import sys
input = lambda: sys.stdin.readline().strip()

import math
inf = math.inf

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def GMI():
    return map(lambda x: int(x) - 1, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''
排序双指针
'''

def solve():
    n, m, k = LII()
    h = LII()
    b = LII()
    h.sort()
    b.sort()
    ans = 0
    i = j = 0
    while i < n and j < m:
        if h[i] <= b[j]:
            i += 1
            j += 1
            ans += 1
        else:
            j += 1
    if ans >= k:
        print("Yes")
    else:
        print("No")


t = 1
for _ in range(t):
    solve()
```

## 第三题：D - Robot Customize https://atcoder.jp/contests/abc431/tasks/abc431_d

**思路：两个思考方向，贪心/dp。贪心是不对的，局部最优解无法推广到全局最优解。这题其实要用差值dp来做，然后发现了一个需要改板子的地方，那就是$inf$变量，如果写$math.inf$会非常慢，为什么？因为$math.inf$是浮点数，每次比较的时候会把非浮点数的数字转化为浮点数进行比较，就会非常慢，因此我们可以改成$10^{18}$，这种大整数，就不会超时了，而且题目基本上都是整数。**

**代码：**
```python
import sys
input = lambda: sys.stdin.readline().strip()

import math
inf = 10 ** 18

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def GMI():
    return map(lambda x: int(x) - 1, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''
差值dp，要仔细学习一下
'''

def solve():
    n = II()

    W = []
    H = []
    B = []
    totW = 0
    for _ in range(n):
        w, h, b = LII()
        W.append(w)
        H.append(h)
        B.append(b)
        totW += w

    # 差值 d ∈ [-totW, totW]，平移到 [0, 2*totW]
    offset = totW
    size   = 2 * totW + 1
    # 初始：dp[0][0+offset] = 0，其余 -inf
    dp = [-inf] * size
    dp[0 + offset] = 0

    for i in range(n):
        w, h, b = W[i], H[i], B[i]
        ndp = [-inf] * size
        for d in range(size):
            if dp[d] == -inf:
                continue
            # 1) 放头：差值 +w
            nd = d + w
            if nd < size:
                ndp[nd] = max(ndp[nd], dp[d] + h)
            # 2) 放身：差值 -w
            nd = d - w
            if nd >= 0:
                ndp[nd] = max(ndp[nd], dp[d] + b)
        dp = ndp

    # 所有 d <= 0 的平移位置就是 [0, offset]
    print(max(dp[:offset+1]))

t = 1
for _ in range(t):
    solve()
```
