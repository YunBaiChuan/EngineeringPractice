## 第一题：1015.可被K整除的最小整数 https://leetcode.cn/problems/smallest-integer-divisible-by-k/description/?envType=daily-question&envId=2025-11-25

**思路：代码实现很简单，但是在理解上就比较困难。一般的想法就是构造大整数1111...1然后来判断其余数，但这个方法显然是会超时的。我们根据mod的性质可知，它的余数必然是在0到n - 1这个闭区间中，那我们就可以每次用mod后的余数来做下一次mod的x，然后本题的条件翻译成数学公式其实就是$x * 10 + 1$，那就把构造大整数1111...1的过程去掉，直接用余数，并且用hash表存储出现过的余数，如果有相同的非空元素说明不行，他会陷入循环节，否则就能满足条件**

**代码：**
```python
class Solution:
    def smallestRepunitDivByK(self, k: int) -> int:
        seen = set()
        x = 1 % k
        while x and x not in seen:
            seen.add(x)
            x = (x * 10 + 1) % k
        return -1 if x else len(seen) + 1
```

## 第二题：圆与三角形 https://www.nowcoder.com/practice/8f13106ae16c4798acb108c820c813e9?channelPut=tracker2

**思路：诈骗题，最重要的是知道一个任意三角形都满足的恒等式
$$
\tan\left(\frac{A}{2}\right)\tan\left(\frac{B}{2}\right) + \tan\left(\frac{B}{2}\right)\tan\left(\frac{C}{2}\right) + \tan\left(\frac{A}{2}\right)\tan\left(\frac{C}{2}\right) = 1
$$
具体来说，它的推导如下，其实就是用到了三角形的恒等变换：**

## 推导：

我们知道三角形中：

\[
\frac{A}{2} + \frac{B}{2} + \frac{C}{2} = \frac{\pi}{2}
\]

令：

\[
x = \frac{A}{2},\quad y = \frac{B}{2},\quad z = \frac{C}{2}
\]

则：

\[
x + y + z = \frac{\pi}{2}
\]

我们要证明：

\[
\tan x \tan y + \tan y \tan z + \tan z \tan x = 1
\]

由于：

\[
x + y = \frac{\pi}{2} - z
\]

所以：

\[
\tan(x + y) = \tan\left(\frac{\pi}{2} - z\right) = \cot z = \frac{1}{\tan z}
\]

另一方面，根据正切加法公式：

\[
\tan(x + y) = \frac{\tan x + \tan y}{1 - \tan x \tan y}
\]

因此：

\[
\frac{\tan x + \tan y}{1 - \tan x \tan y} = \frac{1}{\tan z}
\]

交叉相乘：

\[
(\tan x + \tan y)\tan z = 1 - \tan x \tan y
\]

展开左边：

\[
\tan x \tan z + \tan y \tan z = 1 - \tan x \tan y
\]

移项即得：

\[
\tan x \tan y + \tan y \tan z + \tan z \tan x = 1
\]
