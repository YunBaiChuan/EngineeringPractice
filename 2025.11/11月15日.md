## 第一题：诗超绊 https://ac.nowcoder.com/acm/contest/122725/C

**思路：最重要是观察到d的单调性，这让我们想到二分答案。接下来就是想想$check$函数怎么写，我们又注意到对于这种区间+d问题来说，常常用到差分进行解决。因此我们可以把数组处理成差分数组，这样对于差分数组中的每个$<= 0$的数，我们需要进行每次单个+d的操作，那么我们$check$函数就可以比较操作次数有没有超即可。**

**代码：**
```python
import sys

input = lambda: sys.stdin.readline().strip()

import math

inf = 10 ** 18


def I():
    return input()


def II():
    return int(input())


def MII():
    return map(int, input().split())


def GMI():
    return map(lambda x: int(x) - 1, input().split())


def LI():
    return input().split()


def LII():
    return list(map(int, input().split()))


def LFI():
    return list(map(float, input().split()))


fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''
有单调性，可以进行二分，并且有关于区间的+d问题，我们采用差分优化check函数
具体来说就是差分数组必须每个数都> 0，不满足这个条件的话，就需要进行操作
'''

def solve():
    n, m = MII()
    a = LII()

    flag = True # 优化
    diff = []
    for i in range(1, n):
        if a[i] - a[i - 1] <= 0: # 需要严格大于
            flag = False
        diff.append(a[i] - a[i - 1])

    # 检查是否已经严格递增
    if flag:
        print(0)
        return

    if m == 0:
        print(-1)
        return

    def check(mid):
        op = 0
        for x in diff:
            if x > 0:
                continue
            # op += math.ceil((1 - x) / mid)
            op += (1 - x - 1) // mid + 1 # 上取整优化，比函数快
            if op > m: # 优化
                return False
        return True

    l, r = 1, 10 ** 18
    while l <= r:
        mid = (l + r) >> 1
        if check(mid):
            r = mid - 1
        else:
            l = mid + 1

    print(l if l < 10 ** 18 else -1)


t = II()
for _ in range(t):
    solve()
```

## 第二题：春日影 https://ac.nowcoder.com/acm/contest/122725/D

**思路：首先我们第一反应肯定是$dijkstra$，接下来就是怎么处理题目给出的限制条件，处理好了之后就可以套板子了。注意到，我们应该根据题意，贪心的进行边权的更新，具体来说就是：对于每个节点连接的边权中有个最小值$minw$，我们找到这个$minw$，然后用$fmin(curw, minw * 2)$来进行更新边权，其中$*2$是因为一来一回走了两遍。**

**代码：**
```python
import sys
from heapq import heappop, heappush

input = lambda: sys.stdin.readline().strip()

import math
inf = 10 ** 18

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def GMI():
    return map(lambda x: int(x) - 1, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''
根据题意改边权，然后跑一遍dijkstra，不过板子有点忘了
'''

def solve():
    n, m = MII()
    g = [[] for _ in range(n)]
    for _ in range(m):
        u, v, w = LII()
        u -= 1; v -= 1
        g[u].append([v, w])
        g[v].append([u, w])

    for i in range(n):
        minw = inf
        for j in range(len(g[i])):
            minw = fmin(minw, g[i][j][1])
        for j in range(len(g[i])):
            g[i][j][1] = fmin(g[i][j][1], 2 * minw)

    dis = [inf] * n # dijkstra板子
    dis[0] = 0
    h = [(0, 0)]
    while h:
        d, x = heappop(h)
        if d > dis[x]:
            continue
        for y, dx in g[x]:
            new_dis = d + dx
            if new_dis < dis[y]:
                dis[y] = new_dis
                heappush(h, (new_dis, y))

    print(dis[n - 1] if dis[n - 1] < inf else -1)

t = 1
for _ in range(t):
    solve()
```
