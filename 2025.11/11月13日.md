## 第一题：3228.将1移动到末尾的最大操作数 https://leetcode.cn/problems/maximum-number-of-operations-to-move-ones-to-the-end/description/?envType=daily-question&envId=2025-11-13

**思路：观察性质，可以发现我们要得到最大操作数的话，应该每次移动左边的"1"靠近到右边的"1"，当所有"1"都拼在一块，或者说是连续的时候，此时的操作次数必然是1的次数，因为没有操作空间了。那我们ans = 得到连续"1"的操作次数 + 连续"1"的长度。**

**代码：**
```python
class Solution:
    def maxOperations(self, s: str) -> int:
        p = s.rfind('0')
        if p == -1:
            return 0
        tmp = []
        for i in range(p):
            if s[i] == '1':
                tmp.append(i)
        ans = pre = 0 
        for i in range(1, len(tmp)):
            pre += 1
            if tmp[i] != tmp[i - 1] + 1:
                ans += pre 
        return ans + s[:p].count('1')
```

## 第二题：归并排序 https://www.luogu.com.cn/problem/P1177

**思路：归并排序实际上是先找到中点mid，然后左边排序，右边排序，然后两边merge合并，从而得到整体有序，最终我们不断递归下去，就可以让数组整体有序。这里是没有返回值的写法，不过会更麻烦点对于归并来说**

**代码：**
```python
# 归并牌序
help = [0] * n

def merge(l, mid, r):
    i = l
    a = l
    b = mid + 1
    while a <= mid and b <= r:
        if nums[a] <= nums[b]:
            help[i] = nums[a]
            a += 1
        else:
            help[i] = nums[b]
            b += 1
        i += 1
    while a <= mid:
        help[i] = nums[a]
        i += 1; a += 1
    while b <= r:
        help[i] = nums[b]
        i += 1; b += 1
    for i in range(l, r + 1):
        nums[i] = help[i]

def merge_sort(l, r):
    if l == r:
        return
    mid = (l + r) >> 1
    merge_sort(l, mid)
    merge_sort(mid + 1, r)
    merge(l, mid, r)

merge_sort(0, n - 1)
print(*nums)
```

## 第三题：随机快速排序 https://www.luogu.com.cn/problem/P1177

**思路：随机快速排序本质上来说就是每次在$[l, r]$中随机找到一个数字x，然后把区间分成<=x以及>x两个部分，注意最后x要放在靠近中间的临界点，才能保证中间是我们已经排好的部分，最终递归下去就能排好数组了，但时间复杂度最坏可能到$O(n^2)$，会导致超时，而且随机快排用没有返回值的写法更好写一点。**

**代码：**
```python
# 快速排序
def partition(l, r, x):
    idx = -1
    a = i = l
    while a <= r and i <= r: # 注意是闭区间
        if nums[i] <= x: # 注意这里的代码顺序
            nums[a], nums[i] = nums[i], nums[a]
            if nums[a] == x:
                idx = a
            a += 1
        i += 1
    nums[idx], nums[a - 1] = nums[a - 1], nums[idx]
    return a - 1
    
def quick_sort(l, r):
    if l > r:
        return
    x = nums[l + int(random.random() * (r - l + 1))]
    mid = partition(l, r, x)
    quick_sort(l, mid - 1)
    quick_sort(mid + 1, r)

quick_sort(0, n - 1)
print(*nums)
```
