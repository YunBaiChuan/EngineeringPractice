## 第一题：2536.子矩阵元素加1 https://leetcode.cn/problems/increment-submatrices-by-one/description/?envType=daily-question&envId=2025-11-14

**思路：非常经典的二维差分，不过我忘了板子。主要是注意+1和-1的变量是怎么样的，主要是右上和左下是-1，然后左上和右下是+1，这个主要是依据二维前缀和的计算过程。**

**代码：**
```python
class Solution:
    def rangeAddQueries(self, n: int, queries: List[List[int]]) -> List[List[int]]:
        diff = [[0] * (n + 1) for _ in range(n + 1)]
        for a, b, c, d in queries:
            diff[a][b] += 1
            diff[c + 1][b] -= 1
            diff[a][d + 1] -= 1
            diff[c + 1][d + 1] += 1

        acc = [[0] * (n + 2) for _ in range(n + 2)]
        for i, row in enumerate(diff):
            for j, x in enumerate(row):
                acc[i + 1][j + 1] = acc[i][j + 1] + acc[i + 1][j] - acc[i][j] + x
        
        acc = acc[1:-1] # 这个二维的切片要从[[], []]这种平面形式来看，就是最前面和最后面给去了
        ans = []
        for row in acc:
            ans.append(row[1:-1]) # 这里是每一个里面的最前面和最后面给去了
        return ans 
```
