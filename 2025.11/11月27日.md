## 第一题：3381.长度可被K整除的子数组的最大元素和 https://leetcode.cn/problems/maximum-subarray-sum-with-length-divisible-by-k/description/?envType=daily-question&envId=2025-11-27

**思路：谈到子数组和，就一定要先想到前缀和优化。接着就是另一个很重要的点，当我们枚举前缀和的索引和值，即$j, v$的时候，还需要枚举i，并且保证$j - i$和$k$取$mod$之后为$0$，也就是长度被整除。但这个式子恰好是满足同余定义的，当j与i同余时，两者相减后可以得到为模数k的倍数，因此$mod \ k$之后是为0的。他这个解法非常巧妙，同时引入了$min\_ s$这个数据变量，把$mod \ k$的余数压缩到这个里面，并且不断取对应余数的最小值，那他为什么是有效的？模拟之后发现，因为j是从0开始到最后的，所以说可以取到所有的j % i得到的余数，因此有效。**

**代码：**
```python
class Solution:
    def maxSubarraySum(self, nums: List[int], k: int) -> int:
        pre = [0] + list(accumulate(nums))
        min_s = [inf] * k 
        ans = -inf 
        for j, s in enumerate(pre):
            i = j % k 
            ans = max(ans, s - min_s[i])
            min_s[i] = min(min_s[i], s)
        return ans 
```

## 第二题：小sun的假期 https://www.nowcoder.com/practice/e502f7c4ca9e455ab252d2451f5dacb2?channelPut=tracker2

**思路：注意看数据范围，这题用不了差分，会非常复杂。想想除了差分可以解决区间问题以外，还有什么方法可以解决？——合并区间。合并区间的经典操作就是先按照左端点进行排序，之后扫一遍这个排好序的区间，并且通过$l，r，last$来进行$ans$的计算，用$l - last$，也就是当前的左端点与上一个右端点，如果有空隙，那么得到的就应该是正数，它会计入我们的$ans$中，否则不会。**

**代码：**
```python
import sys

input = lambda: sys.stdin.readline().strip()

import math
inf = 10 ** 18

def I():
    return input()

def II():
    return int(input())

def MII():
    return map(int, input().split())

def GMI():
    return map(lambda x: int(x) - 1, input().split())

def LI():
    return input().split()

def LII():
    return list(map(int, input().split()))

def LFI():
    return list(map(float, input().split()))

fmax = lambda x, y: x if x > y else y
fmin = lambda x, y: x if x < y else y
isqrt = lambda x: int(math.sqrt(x))

'''
这个是区间合并，而不是差分数组，为什么？看数据范围，这应该
算是区间合并的一个经典题目，主要是先按照左端点进行排序，之后
就进行l，r的比较，以此来计算ans
'''

def solve():
    n, m = MII()
    a = [LII() for _ in range(m)]
    a.sort()
    a.append([n + 1, n + 1])
    ans = 0
    last = 0
    for l, r in a:
        ans = fmax(ans, l - last - 1)
        last = fmax(last, r)
    print(ans)

t = 1
for _ in range(t):
    solve()
```
